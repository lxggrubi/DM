Run Instructions:
1.) Double Click DietManager.jar

Guide:
User Tab - Can be used to update name/weight and save
Food Tab - Can be used to add new foods and recipes
Exercise Tab - Can be used to add new exercises
Log Tab - Can be used to log food and exercises for the current date
Daily Values - Can be used to change date and displays current dates statistics
Daily Graph - Displays fat (red), carbs (green), protein (blue) in an automatically updated graph
